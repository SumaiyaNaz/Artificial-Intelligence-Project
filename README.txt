1️- Project Overview
------------------------------------------------------------
This project demonstrates how Constraint Satisfaction Problem (CSP) techniques
can be used to solve the Map Coloring problem efficiently.

The goal is to color regions on a map so that no two adjacent regions share the same color,
using the minimum number of colors (based on the Four Color Theorem).

------------------------------------------------------------
2️- Files Included
------------------------------------------------------------
1. preprocessing.py  → Cleans dataset, builds CSP object, saves as us_county_csp.pkl  
2. map_coloring_solver.py  → Solves the CSP using Forward Checking + MRV algorithm  
3. map_coloring_comparison.py  → Compares 4 algorithms (BT, BT+MRV, FC, FC+MRV)  
4. county_adjacency2010.csv  → Dataset containing county adjacency relationships  
5. Manual.docx  → Detailed project explanation  
6. Presentation.pptx  → PowerPoint slides for presentation  
7. README.txt  → This file

4️- How to Run
------------------------------------------------------------
1. Run `preprocessing.py` first → this creates "us_county_csp.pkl"
2. Then run `map_coloring_solver.py` → to see the colored map output
3. Finally, run `map_coloring_comparison.py` → to compare all algorithms

Each script will show colored map outputs and performance comparisons.
